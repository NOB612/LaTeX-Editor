# LaTeX-Editor
Edit Locally, Compile Remotely 

This is not a open source project. The repository is provided solely for bug reporting and user feedback.

# Downloads

**Windows10/11**: https://apps.microsoft.com/detail/9p1rcn1l1bmh?ocid=webpdpshare

# features:
- **Real-Time Preview**: Instantly view your compiled PDF as you type.
- **Smart Synchronization**: Save your file → auto-trigger remote compilation → see the updated PDF—no manual steps.
- **Forward & Reverse Search**:
     - Ctrl + Click in your source code to jump to the exact spot in the PDF.
     - Click any text in the PDF to return instantly to the corresponding line in your editor.
- **Live Document Outline**: Navigate complex documents with ease using the dynamic structure panel.
- **One-Click Error Navigation**: Click any error or warning in the log to go straight to the problematic line.
- **Full Engine Support**: Works flawlessly with pdflatex, xelatex, and lualatex.

Perfect for students, researchers, and academics who want the power of LaTeX—without the setup hassle.
