LaTeX editor featuring remote compilation and side-by-side preview.

> This is not a open source project. The repository is provided solely for bug reporting and user feedback.

# features:
LaTeX editor featuring remote compilation and side-by-side preview.

Key Features:

* Split-pane Preview: Source code and PDF are displayed side by side; the preview updates automatically after each save.
* Forward Search: double click in the source to jump to the corresponding position in the PDF.
* Inverse Search: Click any text in the PDF to navigate back to the matching line in the source.
* Section Navigation: Browse and jump to document sections (e.g., \section, \subsection) using the outline panel.
* Citation Autocompletion: Typing \cite{ shows available bibliography entries for quick insertion.
* Error Navigation: Click an error in the compilation log to go directly to the problematic line in the source.
* Full Engine Support: Compatible with pdflatex, xelatex, and lualatex.

Ideal for students, researchers, and academics who need reliable LaTeX typesetting without the overhead of local setup.
